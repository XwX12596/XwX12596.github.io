记录利用docker开设Hexo框架简单博客
2023-04-01 21:20:00

准备工作

云服务器

  • 系统：Ubuntu
  • 使用域名：xwxstudio.com

Node.js

从NodeSource安装

或者yay -Sy nodejs

Hexo

官方文档的npm安装。

或者yay -Sy npm

在Node背景下的Hexo安装。

docker

使用官方安装脚本自动安装。

从阿里云用cURL安装：

关于docker-compose的安装可以参考菜鸟教程。

或者yay -Sy docker docker-compose

Hexo配置

文件目录

_config.yml：Hexo的配置文件
themes：放主题文件的文件夹
source/_drafts：放草稿的文件夹
source/_posts：放帖子的文件夹

基本信息配置

详见Hexo配置

Docker配置

Dockerfile

hexo镜像

  • 仅仅作Hexo模板的初始化
  • 运行docker build -t 'hexo' .

blog镜像

  • 生成具体博客镜像
  • 运行docker build -t 'blog' .

docker-compose.yml

  • 在具体文件夹创建容器

  • 开启容器：docker-compose up -d

  • 关闭容器：docker-compose down

写作

修改

  • 直接对宿主机文件进行操作即可实现hexo博客的创建，配置与写作。

文件管理

  • 直接使用Git对本地和云端的数据进行版本管理。
  • 可以使用VScode进行Git的pull和push操作。
